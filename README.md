# Nothing But Beds Pitch Site

Static demo site for the Nothing But Beds redesign pitch.

Live GitHub Pages URL:
https://jack11111.github.io/nothing-but-beds-demo/

Local preview:
Open `index.html` in a browser, or run a small local server from this folder:

```bash
python3 -m http.server 8097
```

## Included

- Mobile-friendly one-page website
- Local HD image assets
- Current offers, locations, phone numbers, hours, email, review score, and product starting prices
- Simple sleep-match interaction
